Mathematics of Language Modelling

What is language model? Why should we learn this? How language modelling is affecting our daily life?
 Before defining language modelling let me define some symbols which we will use to define this.

  1. ν:  set of all words in a language. for example to build a language model in english we might have,
       ν = {the , dog , laugh, saw , bark ,cat , .....}
      We assume ν is finite set. ν can be very large, it can have ten thousand words (even more words are possible).
  2. ν†:  Set of all sentences with the vocabulary ν , this is an infinite set because setences can be of any length.
  3. A sentense in the language is a sequence of words:
      x1x2x3.......xn
    where n ∈ I ; n≥1; and xi ∈ ν ; i ∈ {1,.....(n-1)};
    we assume xn is special symbol STOP.
    Example: the dog barks STOP

  A language model has a finite set ν and a function p(x1,x2,x3,.......,xn) such that :
